# ✍️ AI Handwritten Digit Recognition

An AI-powered Handwritten Digit Recognition System built using **TensorFlow**, **Convolutional Neural Networks (CNN)**, **OpenCV**, and **Flask**. The application allows users to draw handwritten digits on a canvas and predicts the digit in real time.

---

## 🚀 Features

- ✍️ Draw digits on an interactive canvas
- 🧠 CNN-based digit recognition
- 🎯 High prediction accuracy using the MNIST dataset
- 📊 Displays prediction confidence
- 🌐 Flask-based web interface
- ⚡ Real-time inference
- 🖥️ Clean and responsive UI
